# Nifty 50 Stock Market Analysis (2000-2024)

## Overview
Exploratory Data Analysis on 24 years of Nifty 50 stock market data using Python.
The analysis covers market trends, volatility patterns, moving averages and yearly returns.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Findings
- Nifty 50 grew 1260% over 24 years — from 1592 in 2000 to 21666 in 2024
- 2008 was the worst year with -51.8% return (Global Financial Crisis)
- 2009 was the best year with 75.8% return (market recovery)
- 2020 showed highest volatility due to COVID crash
- Golden Cross pattern post 2020 indicates strongest bull run in Nifty history
- Daily returns follow a Normal Distribution — most moves within -1% to +1%

## Analysis Performed
- Closing price trend over 24 years
- Yearly average closing price
- Monthly average analysis
- Daily returns calculation
- Moving averages (50-day and 200-day)
- Yearly volatility analysis
- Yearly returns (best and worst years)
- Distribution of daily returns

## Dataset
Nifty 50 historical data from 2000 to 2024
Source: Kaggle
